# HTML & CSS Practice Project

## Description
This project is created for **practicing HTML and CSS**. The goal is to improve my skills in web development by building simple web pages and experimenting with styling.

## Technologies Used
- HTML
- CSS

## Purpose
- Learn the structure of web pages using HTML.
- Style pages using CSS.
- Practice building responsive and visually appealing layouts.

## How to Use
1. Clone the repository to your local machine.
2. Open the HTML files in your web browser to see the pages.
3. Modify the HTML and CSS to experiment and practice.
